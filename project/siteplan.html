WDD 131 – Dynamic Web Fundamentals

Site Name

Site Name: GreenPath Gardens

Reason: The name represents sustainability, growth, and eco-friendly gardening practices that inspire people to connect with nature.

Optional Domain: greenpathgardens.org

Site Purpose

The purpose of this website is to teach beginners how to start and maintain home gardens. It will provide planting guides, care tips, and seasonal advice to help users grow plants successfully.

Scenarios

  • How can I start a small herb garden indoors?
  • What plants grow best in tropical climates?

Color Schema

#4CAF50 – Primary (Headings and buttons)
#F1F1F1 – Secondary (Backgrounds)

These colors represent freshness and simplicity, matching the theme of nature and gardening.

Typography

Headings: "Poppins", sans-serif

Body Text: "Open Sans", sans-serif

"Poppins" will be used for headings, and "Open Sans" for all paragraph text.
